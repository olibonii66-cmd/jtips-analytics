JTIPS V25 — Estrutura estatística estilo FootyStats

Esta versão reorganiza a análise da partida para seguir o modelo estatístico do FootyStats dentro da identidade/layout JTIPS.

Regra de dados:
1. Usa primeiro os campos diretos da API FootyStats.
2. Quando o campo específico não existe, usa fallback com dados da própria API.
3. Não copia marca, identidade visual ou textos proprietários do FootyStats.
4. As abas foram ampliadas para: Resumo, Quem vai vencer, Gols, Escanteios, Cartões, 1º/2º Tempo, Chutes/NO GOL, Jogadores, H2H, Forma e Partidas.

Arquivos removidos do ZIP:
- READMEs temporários de versões antigas.
- Placeholder de dataset não usado.
